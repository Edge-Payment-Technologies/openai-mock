# OpenAIMock

To start your Phoenix server:

- Run `mix setup` to install and setup dependencies
- Run `mise run mkcert` after installing the `mkcert` binary and trusting its local CA
- Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`openai.test:4900`](https://openai.test:4900) from your browser.

The mock API is available at [`api.openai.test:4900`](https://api.openai.test:4900).

If `openai.test` or `api.openai.test` do not resolve locally, add them to your hosts file so they point at `127.0.0.1`.

Ready to run in production? Please [check our deployment guides](https://hexdocs.pm/phoenix/deployment.html).
